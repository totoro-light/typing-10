# ⌨️ Typing Practice

A free, offline typing tutor for all skill levels — from first-time learners to speed-focused masters.

**Live app → https://totoro-light.github.io/typing-10/**

---

## Features

- **52 progressive lessons** across three categories
  - 🌱 **Beginner** — home row, all keys, numbers
  - 🌿 **Practice** — capitals, punctuation, real sentences
  - ⚡ **Speed** — word sprints, pangrams, quotes, code
- **Visual keyboard** with colour-coded finger guides
- **Animated hand diagrams** showing which finger to use
- **Multi-user** — each player's progress saved separately by name
- **Light / Dark theme** that follows system preference, saved per user
- **Works fully offline** — no server, no account, no tracking

## Privacy

All data is stored only in your browser's `localStorage`. Nothing is sent anywhere.
